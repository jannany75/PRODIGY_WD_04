# 🚀 Personal Portfolio Website

A modern and responsive **Personal Portfolio Website** designed to showcase your skills, projects, education, and achievements as a web developer. Built using **HTML, CSS, and JavaScript**, this portfolio features a clean user interface, beautiful gradients, smooth animations, and a professional layout suitable for internships, placements, and personal branding.

---

## 🌟 Features

### 🏠 Home Section
- Attractive Hero Section
- Professional Introduction
- Short Skills Summary
- Call-to-Action Button
- Modern Gradient Background

### 👨‍💻 About Me Section
- Personal Background
- Educational Information
- Career Objectives
- Professional Overview

### 🛠 Skills Section
- Technical Skills Showcase
- Interactive Skill Cards
- Responsive Grid Layout

### 📂 Projects Section
- Featured Project Display
- Project Descriptions
- Modern Card Design
- Hover Animations

### 📞 Contact Section
- Email Information
- GitHub Profile
- LinkedIn Profile
- Professional Contact Details

### 🎨 UI Features
- Glassmorphism Design
- Blue-Purple Gradient Theme
- Smooth Scrolling Navigation
- Hover Effects & Transitions
- Fully Responsive Design
- Mobile Friendly Layout

---

## 🛠 Technologies Used

- HTML5
- CSS3
- JavaScript
- Google Fonts (Poppins)

## 🎯 Customization

### Update Personal Information

Replace the default content with your:

* Name
* Education
* Skills
* Projects
* Contact Information

### Add Your Projects

Locate the Projects Section and add new project cards:

```html
<div class="project">
    <h3>Project Name</h3>
    <p>Project Description</p>
</div>
```

### Update Contact Details

Replace placeholder links with your:

* GitHub URL
* LinkedIn URL
* Email Address
